#### Libraries

Python 3.6

Pytorch 0.4.0+

Numpy 1.14

Sklearn 0.18

Xgboost 0.81



To run either of the two models, you might need to 

```bash
cd LSTM (cd Ensemble)
mkdir all
mv *.txt all/
```



To run `LSTM` model,

```bash
cd LSTM
python main.py
```

To run `Ensemble` model,

```bash
cd Ensemble
python ensemble.py
```
